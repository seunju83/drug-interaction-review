# 약물 상호작용 검토 (Drug Interaction Review)

허가사항 설명서·식약처 DUR·CYP 대사·약력학 기반 다층 약물 상호작용 예측 도구.

- `index.html` — 메인 앱
- `about_interactions.html` — 데이터 출처·근거 안내
- `_data/drug_interaction_data.js` — 통합 데이터 번들

## 안내
- 약사의 임상 판단을 **보조**하는 도구이며 최종 판단을 대신하지 않습니다.
- 환자 개인정보는 저장·전송하지 않으며, 약물 상호작용만 검토합니다.
- OCR(사진 인식)은 인터넷 연결이 있을 때 브라우저 내에서만 처리됩니다.
